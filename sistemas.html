Sistemas

Sistema de carga de camiones

Las terminales automatizadas requieren que todas las operaciones se realicen a la perfección para garantizar que la carga se realice de acuerdo con los requisitos más estrictos. HOME puede proporcionar sistemas empaquetados completos para aplicaciones de CARGA SUPERIOR o INFERIOR, incluidos todos los medidores, válvulas, filtros, presets y monitores de puesta a tierra y sobrellenado, brazos de carga, transmisores y todo lo necesario para realizar un proceso de carga preciso y seguro para su aplicación única.

Sistema de descarga de biocombustible

Es la solución definitiva que proporciona una eliminación de aire confiable durante todo el proceso de descarga, lo que garantiza la precisión del sistema. El sistema puede mantener un caudal promedio de 120 m3/h durante todo el proceso de descarga, ya sea con tanques de descarga simples o dobles.

El sistema de descarga está diseñado con bombas (principal y secundaria), tanque eliminador de aire, medidor de presión diferencial, válvula de control, filtro, válvula desviadora para calibración del medidor, sistema de monitoreo de puesta a tierra y todo controlado por un preset electrónico Accuload.

Sistema de inyección y bombeo de aditivos

Ofrecemos soluciones personalizadas para sistemas de inyección de colorantes/aditivos en las terminales de distribución, que van desde un inyector de aditivos para una sola línea de producto hasta una solución completa para su terminal con múltiples aditivos y/o marcadores. Podemos proporcionar un sistema completo con patines de bomba, mezcla, inyectores, punto de inyección, etc., para cualquier aplicación.

Unidad LACT

En una medición fiscal la precisión en la misma es lo más importante y debe ser confiable y aceptable tanto para el vendedor como para el comprador. HOME es capaz de proporcionar un servicio de ingeniería completo y personalizado, teniendo en cuenta las necesidades específicas de cada cliente. Podemos suministrar cualquier tipo de tecnología de medición (USM, Corriolis, Turbina, PD Meter, Orifice Fittings, etc.) cubriendo cualquier aplicación tanto para líquidos como para gas.
